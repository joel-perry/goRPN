goRPN is an RPN calculator, with a stack modeled on the HP-16C. It is currently quite minimal, but will eventually adopt more of the functionality of the 16C.

Tips

  • Long-press on any digit to change the display precision
  • Press CLR to clear the entire stack
  • Press CLX to clear the number being entered

goRPN is open-source software. If you would like to contribute, get the source on GitHub
